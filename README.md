# Sparta_Happynewyear
  
스파르타 코딩클럽에서 무료로 공개한 강의 "설맞이 신년운세 패키지"를 수강하여 만든 페이지입니다.  
HTML, CSS, JS를 활용하여 배포하였으나 JS의 경우 강의에서 제공하는 외부 스크립트를 더 많이 사용했습니다.  
배포 또한 강의에서 파일 업로드 시 자동으로 이루어집니다.  
모바일 환경에서도 문제 없이 작동하도록 만들어져있습니다.
